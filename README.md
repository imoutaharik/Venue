Venue Search App

Find the best places to eat, drink near you!

-------------------------------------------------

Really simple to use, search for anything and get a list of venue, the nearest from your current location!

-------------------------------------------------

Technical info : 

Frameworks and libraries used in this project : 

- Reactjs
- Material UI
- React-facebook

Information given : 

- Name of the venue
- Address
- Distance
- Category
